js 与 jquery 入口函数区别

jQuery顶级对象

$

$这个函数根据传入的参数形式不同，会具有不同的功能：

1 选择器功能： $(‘div’);

2 入口函数功能： $(function () {});

3 给DOM对象设置新的功能：$(document).ready(function () {})

选择器

1 标签选择器
$(‘div’).text(‘这是内容’);
2 id选择器
$(‘#box’).text(‘这是box’);
3 类选择器
$(‘.box’).text(‘这是类名为box的元素’);
4 子代选择器
$(‘#list>li’).css(‘backgroundColor’, ‘red’);
5 后代选择器
$(‘#list li’).css(‘backgroundColor’, ‘red’);
交集选择器
获取页面中的类名为box的div
$(‘div.box’).text(‘这是类名为box的div’);
并集选择器
$(‘div,p’).text(‘这是div和p标签’);

过滤选择器

:eq(index) $(‘li:eq(2)’).CSS(‘color’,’red’)
:odd $(‘li:odd’) 奇数行
:even $(‘li:even’) 偶数行
:selected 选中的
:checkbox 选中所有的复选框
:checkbox :checked 选中所有被选中的复选框

筛选选着器

1 children() 子代选择器
children() 不书写选择器参数表示获取所有子元素，如果书写了选择器参数表示选择指定的部分内容
我们发现，这个选择器与 $(‘#list>li’) 的功能是相似的，好处是可以完善链式编程操作。
2 find() 后代选择器
注意：find()如果不设置选择器参数，不会生效

3 parent() 父元素获取,相当于parentNod
由于jQuery对象中可能具有多个DOM对象，所以parent()获取的结果可能为多个元素

4 parents() 祖先元素
parents方法可以不书写参数直接使用，但是没有意义，所以还是推荐设置参数后使用
5 siblings() 同级元素(兄弟元素)

6 next() 下一个 prev() 上一个

7 nextAll()
前面所有同级元素
nextAll

8 eq() 根据指定下标获取元素

DOM对象与jQuery对象

1获取方式

DOM对象：通过DOM的方式获取的标签。
jQuery对象：通过jQuery的方式获取的标签。

2 DOM对象与jQuery对象之间的关系

我们会称jQuery对象是DOM对象的包装集（jQuery对象是一个伪数组，内部包裹了一个或多个DOM对象）

3 DOM对象与jQuery对象使用时的注意点：

DOM对象只能使用DOM的功能，不能使用jQuery的功能
// jQuery对象只能使用jQuery的功能，不能使用DOM的功能

4 DOM对象与jQuery对象的转换方式

将DOM对象转换为jQuery对象：通过$()包裹即可

​ /var box = document.getElementById(‘box’);
$(box).text(‘这是新的内容’);/

将jQuery对象转换为DOM对象：

​ $(‘#box’)[0].innerHTML = ‘这是box的内容’; // 推荐方式
$(‘#box’).get(0).innerHTML = ‘这是更新的内容’;

获取元素

获取多个标签中下标为奇数的元素

$(‘li:odd’).text(‘这是下标为奇数的元素’);

获取多个标签中下标为偶数的元素

$(‘li:even’).text(‘这是下标为偶数的元素’);

获取下标为多少的元素

$(‘li:eq(2)’).text(‘这是第三个li’);
