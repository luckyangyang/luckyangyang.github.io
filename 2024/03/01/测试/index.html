javascript API笔记

获得元素document.

getElementById

var p1 = document.getElementById( ‘p1’ );
语法: document.getElementById( ‘id’ )
习惯: 写标签的时候, 属性使用 双引号, 写字符串的时候属性写 单引号

getElementsByTagName

document.getElementsByTagName( ‘p’ );

getElementsByTagName

var div = document.getElementsByTagName( ‘div’ )[ 0 ];

querySelector

// 选择器 ( selector )
// -> 基本选择器: #id, .className, tagName, *
// -> 复合选择器: 子代选择器( > ), 后代选择器( ‘ ‘ ), …

    // 如果想到定位 这两个 p
    // #dv1 > p
    
    // 使用选择器获得元素的 语法
      // docuemnt.querySelector( '选择器的字符串' ) 获得所有的 符合选择器的元素中的 第 0 个

querySelectorAll

// document.querySelectorAll( ‘选择器字符串’ ) 获得所有的 符合选择器的元素( 伪数组 )

document.body

如果需要获得页面中的 body, 除了使用 get 或 query 系列的方法之外
还可以使用 document.body, 其就是 当前页面的 body 标签
