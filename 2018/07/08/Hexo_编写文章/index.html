Hexo 编写文章并部署

typora 软件 .md文件编辑器

文章文件位置

将写好的md文件放到 Hexo\hexodemo\source_posts 目录下

常用快捷键

ctrl + alt + z 插入代码
ctrl + shift + i 插入图片
ctrl + alt +u 插入点
ctrl + shift + q插入文字详情块
ctrl + 1,2,,3,4,5 标题

编辑好后
Hexo\hexodemo 目录下
shift + 鼠标邮件 打开 git bash
